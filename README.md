# Plugging into the Future: An Exploration of Electricity Consumption Patterns

## Project Overview
This project analyzes electricity consumption patterns across different states and regions using data visualization techniques. The goal is to explore trends, identify high-consumption areas, and understand how electricity usage varies over time.

The project uses **Tableau dashboards** to visually represent electricity distribution, consumption trends, and geographical patterns.

Electricity consumption data analysis is important for understanding energy demand, improving planning, and supporting sustainable energy management.

## Objectives
- Analyze electricity consumption across different states
- Identify top electricity consuming states
- Visualize electricity usage trends over time
- Explore regional electricity distribution
- Present insights using interactive dashboards

## Tools & Technologies
- Tableau Public
- Data Visualization
- Excel / CSV Dataset
- GitHub for version control

## Dataset
The dataset contains electricity consumption information including:

- State names
- Regions
- Monthly electricity usage
- Yearly electricity consumption
- Geographic coordinates for mapping

The data is used to generate visual insights through Tableau dashboards.

## Dashboard Visualizations

### 1. Region Electricity Distribution
A pie chart showing electricity consumption across different regions such as:

- Northern Region
- Western Region
- Southern Region
- Eastern Region
- North Eastern Region

### 2. Electricity Usage Trend
A line chart showing electricity consumption trends across different years.

This helps analyze whether electricity consumption is increasing or decreasing over time.

### 3. Electricity Consumption by State
A bar chart comparing electricity usage across different Indian states.

This helps identify the states with the highest and lowest electricity consumption.

### 4. Electricity Consumption Map
A geographic map showing electricity usage across India using location markers.

This visualization helps understand regional distribution of electricity demand.

### 5. Top 10 Electricity Consuming States
A horizontal bar chart highlighting the top states with the highest electricity usage.

### 6. Monthly Electricity Consumption Trend
A multi-line chart showing monthly electricity consumption patterns.

This helps identify seasonal variations in electricity demand.

## Key Insights
- Some states have significantly higher electricity consumption due to industrial activity and population.
- Electricity usage shows seasonal variations during certain months.
- Regional differences indicate uneven electricity demand distribution across the country.
